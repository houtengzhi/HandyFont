### 集成

在项目根目录下的build.gradle文件中添加
```
allprojects {
    repositories {
        jcenter()
    }
}
```

在mudule下的build.gradle文件中添加
```
implementation 'com.yechy.handyfont:handyfont:1.0.1'
```
